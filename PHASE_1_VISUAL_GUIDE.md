## 🎯 Responsive Design

### Desktop (1200px+)
```
Hero:         2-column layout (text + visuals side-by-side)
Value Cards:  4-column grid
Mode Cards:   3-column grid
All effects:  Hover states active
```

### Tablet (768px-1199px)
```
Hero:         2-column but tighter
Value Cards:  2-column grid
Mode Cards:   2-column grid
Touch areas:  Minimum 48x48px
```

### Mobile (<768px)
```
Hero:         1-column stacked
Value Cards:  1-column stacked
Mode Cards:   1-column stacked
Buttons:      Minimum 48x48px tap targets
Font:         Minimum 16px (no zoom needed)
```

---

## ✨ Interactive Elements

### Hover State Changes
```
Buttons:     Color shift + translate up (-4px) + shadow grow
Cards:       Border color change + translate up (-8px) + shadow
Links:       Color fade or underline appear
Images:      Subtle scale or opacity transition
```

### Click Actions
```
"Sign In":           → Navigate to login page
"Get Started":       → Navigate to login page
"Start Assessment":  → Navigate to login page
"Learn How":         → Smooth scroll to #how-it-works
Mode Cards:          → Expand/collapse details
```

## 🔍 Accessibility

### Visual
- Color contrast: 4.5:1 or higher
- Font sizes: minimum 16px for body
- Touch targets: minimum 48x48px
- No icons without text labels

### Semantic
- Proper heading hierarchy (h1, h2, h3)
- Semantic HTML button, nav, section tags
- Form labels associated
- Focus states visible

### Screen Reader
- Alt text structure
- ARIA labels where needed
- Skip navigation link
- Announce interactive states

**Overall Tone**: Professional yet approachable, confident but not arrogant

Headlines:     Clear, benefit-focused, action-oriented

## 🎯 Conversion Elements

### Trust Signals
✅ Specific numbers (4 archetypes, 12 weeks, 7 tones)
✅ Clear methodology (3-step process)
✅ AI credibility (Claude Sonnet 4.5)
✅ Low commitment ("5 minutes")

### Objection Handlers
```
"Why should I trust this?"
→ See the process, archetypes, outcomes

"Is this going to waste my time?"
→ "Results in 5 minutes" + "Free"

"How is this different?"
→ Value props + learning modes section

"Will this actually help?"
→ Outcomes section shows clear benefits
```
